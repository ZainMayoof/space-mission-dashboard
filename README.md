# ✨ Space Mission Progress Dashboard

isplays the stages of a space mission from launch to orbit, docking, and landing. It’s designed to help interns, students, and educators understand how complex NASA missions unfold step by step.

## ✨ Purpose

This dashboard helps NASA and its partners by:
- Visualizing mission stages in a clear, interactive way
- Supporting training
- Providing a quick reference/conceptual tool for mission monitoring

- For example, An astronaut candidate or NASA intern can use this dashboard to:Familiarize themselves with the mission structure and simulate the progression of a mission

## ✨ Technologies Used

- HTML, CSS, JavaScript
- [Chart.js](https://www.chartjs.org/) (for mission timeline visualization)
- Miscrosoft Power BI
- GitHub for project management, collaboration, and version control

## ✨How to run
- ensure you have git installed
-  open terminal and clone: git clone https://github.com/your-username/space-mission-progress-dashboard.git
- Navigate to the folder: cd space-mission-progress-dashboard
- Open the folder, and click on index.html.
- You can also just download the ZIP File
